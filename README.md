# thrift-java-android-ios-example
Example project to show hot to use thrift on Java(Server, Android) and iOS

Server: <br/>
SpringBoot 1.2.0 <br/>
Thrift 0.9.1 <br/>
Maven <br/> <br/>

To start server use: java -jar builds/thrift-server-1.0-SNAPSHOT.jar <br/> <br/>

Android: <br/>
Min SDK 14
ppcompat-v7 22.2.0
libthrift 0.9.1
jsr250-api 1.0 <br/><br/>
